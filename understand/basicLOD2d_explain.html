basicLOD2d.m 逐段讲解

目标：参照你现有解释页的风格，把 basicLOD2d.m 的主脚本与子程序串成一条完整“建模→离散→校正→求解→对照→误差”的解题链路。

这段代码在解决什么问题

二维椭圆扩散方程（Dirichlet 边界）： -∇·(A(x)∇u) = f, x ∈ Ω=(0,1)^2 u = 0, x ∈ ∂Ω 其中 A(x) 在细尺度上快速变化（随机 0/1 相位材料）， 代码使用 LOD（Localized Orthogonal Decomposition）在粗网格上构造多尺度基， 并与细网格参考 FEM 解进行比较。

一句话：用“局部校正过的粗网格基函数”逼近原本需要细网格才能分辨的多尺度扩散解。

输入与输出

输入（核心参数）

alpha, beta Neps Hlevel, hlevel ell Aeps, A(x), f(x)

Hlevel 控制粗网格，hlevel 控制细网格，ell 控制局部 patch 半径。

输出（数值结果）

uHms uH uh energy/L2 error

uHms：LOD 细尺度重建解；uH：LOD 粗自由度系数；uh：细网格参考解。

解题路径（按算法流程）

  1. 构造随机高对比扩散系数 A，并设定右端 f。
  2. 生成粗网格 TH 与细网格 Th，并建立粗细网格之间的限制/延拓映射。
  3. 构造准插值算子 IH，以及元素 patch 邻接关系。
  4. 逐个粗单元求局部校正子 CT{k}，再拼成全局校正算子 C_ell。
  5. 得到 LOD 基函数矩阵 G=P1-C_ell，组装并求解粗尺度 LOD 线性系统。
  6. 重建细尺度 LOD 解 uHms，并计算细网格参考解 uh。
  7. 可视化 A、uh、uHms、uH，并打印能量范数和 L2 误差。
  8. 子程序部分提供网格加密、边提取、DG 刚度装配、面积计算等基础工具。

关键行深讲（与原脚本对应）

  • 39：把离散系数矩阵 Aeps 包装成点值函数 A(x)，实现“系数场采样”。
  • 95-100：构造 PiHdg, EH, IH，这是 LOD 正交分解里“粗空间投影/准插值约束”的核心。
  • 106-116：用稀疏邻接矩阵构造 ell-patch，决定校正子局部化范围。
  • 157-160：解局部 saddle-like 系统并消元得到单元校正子 CT{k}。
  • 184-186：从局部校正子拼全局校正算子 C_ell，得到 LOD 基 G=P1-C_ell。
  • 196-203：组装 coarse LOD 系统并重建细尺度解 uHms。
  • 209-212：求细网格参考 FEM 解 uh，用于误差评估。
  • 267-271：输出能量范数和 L2 误差，直接给出方法效果量化指标。

子程序职责地图

  • refineMesh (282-300)：重复调用 refine，返回细网格与一组跨层映射矩阵（P1,R1,P0,P1dg）。
  • refine (305-330)：红加密每个三角形，更新边界节点和网格连接关系。
  • computeEdges (335-342)：提取网格唯一边集合及边重数（判断边界/内部边）。
  • assembleStiffnessMatrixDG (349-371)：基于单元梯度与系数装配 DG 刚度矩阵。
  • computeArea (376-380)：计算每个三角形面积，供质量/刚度构造使用。
